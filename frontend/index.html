    ℹ️ About This Project

    📁

    Dataset

    Pima Indians Diabetes Dataset — 768 patient records with 8 clinical features sourced from the National Institute of Diabetes and Digestive and Kidney Diseases.

    🌲

    ML Model

    Random Forest Classifier with 100 estimators, max depth 10, balanced class weights. Compared against Logistic Regression for rigorous evaluation.

    ⚙️

    Tech Stack

    FastAPI (Python) · SQLite · SQLAlchemy · scikit-learn · Chart.js · Vanilla HTML/CSS/JS

    🔍

    Features

    Real-time diabetes risk prediction, prediction history tracking, interactive analytics charts, feature importance visualization, and multi-model metric comparison.
